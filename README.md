# Сборка контейнера с приложением Python 

1. Исходный код в папке /flask
2. В качестве базового образа использовать
   -  Python 3.9.25
   -  Alpine 3.22
3. Установка зависимостей через __pip3 install__

# Двухэтапная сборка контейнера c приложением GoLang

1. Исходный код в папке /gateway
2. Для компиляции нужно использовать контейнер с версией GoLang 1.24.2
3. В контейнер необходимо скопировать:
   - ./gateway/main.go
   - ./go.mod
4. Установка зависимостей командой __go mod tidy__
5. Компиляция приложения __go build -o gateway__
6. Для сборки итогового контейнера использовать образ alpine 3.22
